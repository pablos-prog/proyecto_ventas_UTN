## 🎯 Objetivo del proyecto

Este proyecto tiene como objetivo aplicar control de versiones con Git y GitHub, junto con análisis de datos utilizando Python en Google Colab.

Se simula una empresa que analiza sus ventas para obtener indicadores de rendimiento.

---

## 📁 Estructura del repositorio

datos/
scripts/
resultados/
README.md

---

## 📊 Dataset utilizado

El dataset contiene información de ventas con las siguientes columnas:

- fecha
- producto
- cantidad
- precio

---

## ⚙️ Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Git / GitHub
- Google Colab

---

## 📈 Análisis realizado

- Cálculo de ventas totales
- Producto más vendido
- Ventas por producto
- Generación de gráficos de ventas

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio desde GitHub
2. Abrir en Google Colab
3. Ejecutar el script principal:

scripts/analisis_ventas.py
